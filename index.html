Square math tests

Приложение для проведения тестирования по квадратным уравнениям

Общая информация.
Предназначение: проведение тестов на ПК на тему квадратных уравнений (школьная программа), задания фиксированы, всего 6 вариантов, после запуска псевдослучайным образом выбирается один. Локализация украинская, создавалась для украинской аудитории.

Техническая информация.
Тесты состоят из двух частей - самих тестов и сервера для сбора результатов. Оба приложения независимы. Для обмена информацией используется 4455 порт. При запущеном сервере в него будут поступать данные о учениках, времени начала\конца, состоянии и оценках после завершения тестов. Настройка приложений-тестов для нормальной работы сервера сводится к редактированию файла host.con в директории программы (по умолчанию C:\Program Files\mathTest_square\host.con) с указанием IPv4 адреса ПК на котором запущен сервер. В случае отсутствия файла host.con - он генерируется автоматически при каждом запуске приложения-тестов с адресом 127.0.0.1. Так же производится логирование всех действий в файл log.txt .

О разработке.
Используется Qt5, в качестве инсталятора выбран Inno Setup. Остальное - в исходниках :) .
